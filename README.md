# raft-secure-transport

Secure raft transport layer via TLSv1.3 with pinned Ed25519 certificates for [`hashicorp/raft`](https://github.com/hashicorp/raft).

> [!IMPORTANT]
> This project is currently in development and not yet ready for production use.

## Overview

This repository contains work-in-progress code for implementing a secure transport layer using TLSv1.3 with ed25519 certificates and pinned certificates between nodes.

## Getting Started

Documentation and usage examples will be added as the project develops. An initial example for testing has been implemented in [`./examples/hello-world/main.go`](./examples/hello-world/main.go).
